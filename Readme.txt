## Installation

1. Ensure you have [Pipenv](https://pipenv.pypa.io/en/latest/) installed:
   ```sh
   pip install pipenv

2. Install dependecies

    pipenv install

3. Activate the virtual environment

    pipenv shell

Alternatively, if you prefer not to use Pipenv:
```sh
pip install -r requirements.txt
